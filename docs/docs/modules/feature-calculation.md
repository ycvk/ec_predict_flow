# 特征计算

特征计算模块用于计算特征因子并为涨跌事件设置真实值标签，为模型训练提供输入数据。

## 功能说明

- 基于下载的K线数据计算特征因子,保存到特征文件中
- 生成预测标签,保存到标签文件中
- 数据预处理和清洗

## 特征计算使用步骤

1. 确保已完成数据下载
2. 进入"特征计算"模块
3. 选择要处理的数据文件,这些文件是数据下载模块下载好的
4. 选择要计算的特征类型,以下特征均为量价特征
   - alpha158:来自微软开源项目qlib的特征因子集合
   - alpha216:将alpha158的部分特征进行扩展得到的特征因子集合
   - alpha101:WorldQuant发表的论文《101 Formulaic Alphas 》 ，其中公式化地给出101个alpha因子
   - alpha191:国泰君安证券研究者,在《数量化专题: 基于短周期价量特征的多因子选股体系》研报中提出的191个因子
   - alpha_ch:作者让大模型生成的包括20多类共176个的特征因子集合(效果一般,待完善)
5. 点击"计算特征"按钮
6. 等待计算完成,特征计算可能需要较长时间,十万行数据可能需要十几分钟

## 标签计算使用步骤
1. 确保已完成数据下载
2. 选择要处理数据文件
3. 选择标签类型和过滤指标
   - 标签类型: 包括上涨和下跌,选择上涨时,只标注上涨的k线,这样训练的模型只用于判断上涨,训练难度更低
   - 过滤指标: 选择过滤指标,包括rsi或cti,用于过滤单边行情,只有在满足过滤指标超卖或超买时才开仓并设置标签
   - 标签窗口: 本系统的标签计算模块采用了计算滑动窗口内涨跌k线的比例来作为真实标签值
   - 预测周期: 预测的事件长度,如果要预测10分钟后的涨跌,那么设置为10

## 输出文件

计算结果保存在backend文件夹的 `data/processed/` 目录下。

## 注意事项

- 特征计算可能需要较长时间
- 确保有足够的内存空间
