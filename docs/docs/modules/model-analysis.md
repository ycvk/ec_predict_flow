# 模型分析

模型分析模块使用代理模型筛选关键特征并直接得到判断阈值，得到策略的最终的交易规则。

## 功能说明

- 使用代理模型简化复杂模型
- 筛选最重要的特征
- 确定特征阈值
- 生成可解释的交易规则

## 使用步骤

1. 确保已完成模型解释
2. 进入"模型分析"模块
3. 选择关键特征，右侧可以同时查看当前特征的解释图
4. 点击"训练代理模型"按钮
5. 代理模型训练完成后得到多条决策规则,每条决策规则具有明确的判断条件
## 代理模型

代理模型是对原始复杂模型的简化：
- 使用决策树作为代理
- 保留最重要的特征
- 生成简单的决策规则

## 输出结果

- 关键特征列表
- 决策规则

## 应用价值

- 简化策略实现
- 提高可解释性
- 降低计算复杂度
- 便于人工审核
