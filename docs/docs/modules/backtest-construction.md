# 构建回测

构建回测模块根据筛选的特征和阈值构建回测策略。

## 功能说明

- 基于模型分析结果自动构建交易策略
- 基于交易策略执行历史回测

## 使用步骤

1. 确保已完成模型分析
2. 进入"构建回测"模块
3. 选择特征数据文件
4. 点击"开始回测"按钮
5. 查看回测结果，可以查看
   - 胜率和收益
   - 使用的决策规则
   - 资金曲线

## 回测参数

- **初始资金**: 回测起始资金量
- **盈利金额**: 每次预测成功的收益
- **亏损金额**: 每次预测成功的损失
- **未来k线数**: 用来控制预测的时间长度

## 回测指标

- 获胜次数
- 失败次数
- 胜率
- 最大回撤

## 输出结果

- 胜率和收益
- 使用的决策规则
- 资金曲线

## 注意事项

- 回测结果仅供参考
- 注意过拟合风险
- 考虑实际交易成本
- 进行样本外验证
