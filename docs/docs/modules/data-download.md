# 数据下载

数据下载模块用于下载K线数据并保存为pkl格式，是整个工作流程的第一步。

## 功能说明

- 下载指定交易对的历史K线数据
- 支持多种时间周期
- 数据保存为pkl格式便于后续处理

## 使用步骤

1. 在工作流概览页面点击"数据下载"模块
2. 配置下载参数：
   - 交易对（如：BTCUSDT）
   - 时间周期（如：1h, 4h, 1d）
   - 起止时间
3. 代理设置: 需要填写下载k线时的代理服务器地址，如：http://127.0.0.1:10808
4. 点击"开始下载"按钮
5. 等待下载完成，系统会显示下载进度

## 输出文件

下载的数据将保存在backend文件夹的 `data/raw/` 目录下，文件名格式为：`{symbol}_{interval}.pkl`

## 说明
数据下载功能用于将预设时段的k线保存为文件,用于后续计算特征和打标签
可以通过选择起止时间下载不同持续时间短的k线数据
建议下载两段数据,数据长度为8:2,可以用于建立训练集和测试集

## 注意事项

- 确保网络连接正常
- 下载可能需要较长时间
- 建议先下载小范围数据进行测试
